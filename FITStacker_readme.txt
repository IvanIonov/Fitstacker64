----------------------------------------------------------------------------
Кнопки основного окна - 

Add... 		- добавить кадры в список

Del    		- удалить кадр из списка
Shift+Del 	- очистить весь список
Ctrl+Del  	- удалить кадр из списка И С ДИСКА (для бракованых кадров)
Ctrl+Shift+Del  - удалить все кадры с диска и очистить список (я так удаляю флеты после создания мастер флета)

List...		- выдать список всех кадров и их Сигнал/Шум (если выделены 2 квадрата), полезно для оценки качества кадров, выкидывания явного брака, а так же для оценки результатов работы разных программ сложения.

Prev и Next (стрелки на клавиатуре влево-вправо) - листать список кадров

View 1:1	- включает 100% просмотр
Gamma 		- сжатие гистограммы логарифмом (влияет только на просмотр)
на цветных кадрах сжимает яркость с сохранением насыщености цвета

Save 		- сохранить показываемый FITS в формате Float 
	+Shift - в формате uint 16 (0..65535)
	+Ctrl  - в формате int 16 (-32768..32767)

Export TIF	- сохранить 16-битный тиф растянутой картинки просмотра

----------------------------------------------------------------------------
Работа мышкой в основном окне - 

левой кнопкой - выделяется зеленый квадрат с фоном 
правой кнопкой - выделяется красный квадрат с объектом (он должен быть достаточно яркий но не пересвеченый)

!!если квадраты не выделены, сложение будет простое, не адаптивное, без учета коэфициентов сигнал/шум
Без адаптивки я обычно складываю флеты, дарки и лайты. 

Ctrl + левая кнопка - таскать изображение по окну (в 100% просмотре)


----------------------------------------------------------------------------
Окно сложения - 

Ignore Black - не учитывать пиксели темнее чем заданное число
После выравивания, у кадров появляются черные поля, 
чтобы их отбросить нужно задать число Ignore Black меньше фона кадра, но больше 0. 

Ignore White - не учитывать пиксели светлее чем заданное число
Сделано для сложения коротких и длинных экспозиций вместе, 
чтобы пересвеченные на длинных выдержках области (звезды) брать с кадра короткой выдержки.

Blend White - ширина плавной границы отбрасывания пересвеченных пикселей.
сделана чтобы сгладить переход между кадрами коротких и длинных экспозиций.

View - 
	Image 		- текущий кадр
	Deviation 	- карта отклонений в каждом пикселе (суммарная карта всех дефектов по всем изображениям)
	Clipping 	- показывает цветом, что будет отброшено сигмой при сложении (красные - слишком яркие, синие слишком темные)
В этом режиме удобно настраивать величину Sigma, 
критерий такой - отбрасываться должны пиковые выбросы (треки частиц, спутники), а не равномерный шум.

----------------------------------------------------------------------------
Background 	- оценка неравномерности фона каждого кадра 
Normalize Area 	- размер квадрата осреднения при оценке фона.
Iterations 	- количество итераций при выкидывании звезд из оценки фона. 
Show Bgr 	- синтезировать и показать фон для текущего кадра
Splined		- включает интерполяцию фона сплайнами (вместо линейной)

Кнопка Grad. Бывают кадры, где черного фона практически нет, а градиент убирать надо.
Алгоритму достаточно выделить 3 области с фоном в разных концах кадра (не на одной линии), по нажатию кнопки Grad программа посчитает по 3-м областям плоский градиент и вычтет из кадра.

Если выделить 4 или больше областей фона, то работать будет чуть более сложный алгоритм, вычитающий фон "пятнами" вокруг каждого прямоугольника. Размер пятна берется из параметра Normalize Area. 

----------------------------------------------------------------------------

Sigma Clip - включает сигма клиппинг, без этой галки будет просто адаптивное суммирование. 
Sigma Fade - позволяет не сразу отбрасывать значение за пределом сигмы, а плавно понижать вес значения 
по мере удаления от порога. 

Sigma Fade = 0 - это будет стандартный сигма клиппинг. 

Local Sigma Scale - понижение (повышение) сигмы для выбранного файла, помогает настроить . 

Create - запуск сложения

----------------------------------------------------------------------------
Окно списка файлов - 
To Iris - именует все файлы для ириса name1.fit, name2.fit и т.д. первым в списке будет активный кадр
нормирует диапазон значений в пределы 0...32767
если выбраны 3 области фона, то убирает градиент фона на каждом кадре

Rescale All - приводит яркость фона и яркость объекта всех кадров к активному. пересохраняет все файлы добавляя в имена "_sc"

Substract All - вычитает выбранный кадр из всех остальных. С подбором яркости у каждого файла по выделенным областям.
пересохраняет все файлы добавляя в имена "_sub"
Мне это было нужно для вычитания звезд из каждого отдельного кадра с кометой. Так же полезно для вычитания звезд из узкополосных кадров для отдельной обработки звезд и туманности.
